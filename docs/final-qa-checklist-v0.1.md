# 最終QAチェックリスト v0.1

実施日: 2026-02-27  
対象: きょうの体調メモ（Webデモ）

## 0. 事前準備
- `npm run check` が通る
- `npm start` で起動
- `http://localhost:8787/` を開く

## 1. 基本入力
- [ ] 症状チップを押すと症状欄に反映される
- [ ] スライダー値と数値バッジが連動する
- [ ] 「この内容で記録する」で保存できる
- [ ] 保存後に「今日の記録: 完了」へ変わる

## 2. AI表示
- [ ] 「まとめ更新」で今日のまとめが表示される
- [ ] 「傾向更新」で傾向が表示される
- [ ] 「次の一歩更新」で提案が表示される
- [ ] 「相談先候補を表示」で候補JSONが表示される

## 3. 音声機能
- [ ] 「音声スタート」でメモ追記ができる
- [ ] コマンドモードで「つらさ7」「気分3」が反映される
- [ ] 「保存して」で音声保存が実行される
- [ ] 音声候補ボタンで再適用できる

## 4. 安全機能
- [ ] 安全チェックに通常文を入れると low/medium が出る
- [ ] 「胸が強く痛いし息が苦しい」で high が出る
- [ ] high時に強調表示（赤系）になる

## 5. 継続支援
- [ ] 最近の記録に新規保存分が表示される
- [ ] 月カレンダーに記録日が表示される
- [ ] 連続記録日数・総記録が更新される

## 6. 共有
- [ ] 「共有メモをコピー」でコピーできる
- [ ] 「内容をプレビュー」で共有文を確認できる

## 7. 完了条件
- [ ] 上記すべてOK
- [ ] 致命的な不具合なし
